# Project Context: Form & Banner SaaS Builder

이 프로젝트는 사용자가 직접 입력 폼과 이미지 배너를 구성하여 고유 링크를 생성할 수 있는 SaaS 플랫폼입니다.

## Tech Stack
- **Frontend**: Next.js 14+ (App Router), TypeScript
- **Styling**: Tailwind CSS, Shadcn UI (UI 컴포넌트 라이브러리)
- **Backend/Auth/DB**: Supabase
- **Deployment**: AWS (Amplify or App Runner)

## Core Features
1. **User Dashboard**: 생성한 프로젝트 리스트 확인 및 관리.
2. **Form Builder**: 
   - 사용자가 텍스트, 선택지 등 필드를 추가/삭제/수정.
   - 이미지 배너 업로드 및 미리보기.
3. **Dynamic Rendering**: `/[slug]` 경로를 통해 생성된 폼 페이지 실시간 렌더링.
4. **Data Collection**: 외부 유저가 제출한 응답을 저장하고 제작자에게 통계 제공.

## Rules & Standards
- 모든 컴포넌트는 `components/` 폴더 내에 기능별로 세분화한다.
- Server Component와 Client Component를 명확히 분리하여 성능을 최적화한다.
- 데이터 Fetching은 Supabase 클라이언트를 사용하며, 타입 안정성을 위해 `database.types.ts`를 준수한다.
- 디자인은 모던하고 깔끔한 화이트 톤의 대시보드 스타일을 유지한다.

## Instructions for AI
