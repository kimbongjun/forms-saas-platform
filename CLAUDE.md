# 마케팅 프로젝트 관리 플랫폼 — Claude 컨텍스트

## 스택
- Next.js 16 App Router · TypeScript · Tailwind CSS v4 · Supabase · Resend · @dnd-kit · Tiptap v2 · Zustand

## 핵심 규칙
- `params`는 반드시 `await` (`params: Promise<{id:string}>`)
- Server Component → `createServerClient()` / Client Component → `createClient()` / 공개 폼(비인증) → `createPublicClient()`
- Tiptap: `dynamic(...,{ssr:false})` + `immediatelyRender: false` 필수
- 타입 정의 위치: `src/types/database.ts`

## 향후 실행
- [x]**프로젝트 상세 메뉴생성** : 프로젝트별 상세에서 해당 프로젝트의 산출물과 인게이지먼트(outputs)를 관리하기 위한 메뉴와 프로젝트 전체 운영결과를 볼 수 있는 메뉴(insights) 생성해줘. 이해를 돕기위한 트리구조를 참고와 와이어프레임 참고해서 제작해줘. (와이어프레임은 참고용이니 현재 구조에 맞게 진행해도돼.)

세부 기능 : api를 연동하여 데이터를 가져오는 경우 1시간 단위로 갱신해주고, 수동 새로고침을 위한 새로고침 버튼을 배치해서 관리자가 갱신이 필요할때 갱신되었으면 좋겠어.

```
└── /projects/[id]
    ├── /outputs (신설: 프로젝트 산출물 & 인게이지먼트)
        ├── /deliverables (인스타/유튜브 영상 등 개별 산출물 리스트 및 실시간 지표)
        └── /clippings (보도자료 및 외부 링크 아카이빙)
    └── /insights (신설: 프로젝트 전체 운영 결과 보고서)
        └── (대시보드 형태) (SNS 합산 지표, 목표 달성률 시각화)
```
